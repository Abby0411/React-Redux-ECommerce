# 🛒 React Redux E-Commerce Website

A modern and responsive **E-Commerce Web Application** built using **ReactJS** and **Redux**, focusing on delivering a seamless shopping experience through a clean, intuitive, and responsive user interface.

---

## 🚀 Features

- 🛍️ Browse a wide range of products
- 🔍 View detailed product information
- 🛒 Add and manage items in the shopping cart
- 🔄 State management using Redux
- 📱 Fully responsive design for desktop and mobile devices
- ⚡ Fast and optimized performance with Vite
- 🎨 Clean and user-friendly interface

---

## 🛠️ Tech Stack

- ReactJS
- Redux
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/Abby0411/React-Redux-ECommerce.git
```

### Navigate to the Project Directory

```bash
cd React-Redux-ECommerce
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```
